# 開発計画・改善計画

このドキュメントでは、GenSou AI の今後の開発計画、機能追加計画、および改善提案を管理します。

## 実装予定の機能

### 優先度：高

1. **AI アシストパネルの強化**
   - 生成結果のカーソル位置挿入機能
   - 文脈に応じたプロンプトテンプレート
   - 過去の生成履歴の保存と再利用

2. **エクスポート機能の拡張**
   - PDF エクスポート（縦書き対応）
   - EPUB エクスポート（電子書籍フォーマット）

3. **ファインチューニング機能**
   - ユーザー固有のスタイル学習
   - キャラクターの口調学習

### 優先度：中

4. **バージョン管理機能**
   - 小説の変更履歴
   - 過去バージョンの比較・復元

5. **コラボレーション機能**
   - 複数ユーザーでの共同執筆
   - コメント・提案機能

6. **クラウド同期**
   - データのクラウド保存
   - デバイス間でのデータ同期

### 優先度：低

7. **モバイルアプリ最適化**
   - ネイティブアプリ化の検討
   - タッチ操作の最適化

8. **オフライン対応の強化**
   - Service Worker の改善
   - ローカルストレージの活用

## 改善提案

### パフォーマンス改善

- マインドマップのレンダリング最適化（大規模ノード対応）
- エディタの読み込み速度向上
- AI API レスポンスのキャッシング

### UX 改善

- オンボーディングチュートリアルの追加
- ショートカットキーの充実
- ダークモードのカスタマイズ機能

### 技術的改善

- テストカバレッジの向上
- CI/CD パイプラインの構築
- ロギング・監視システムの導入
